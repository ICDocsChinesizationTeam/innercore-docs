Optional damage

damage: number

Optional isNative

isNative: boolean
  • Defined in core-engine.d.ts:11470

If true, the tool is vanilla Minecraft tool. Generally used within Core Engine code and should not be used within mods until you really know what you're doing

Optional isWeapon
